# 채팅 줄 간격 조절

> Languages: [English](./en.md) · **한국어**
>
> Related: [#218](https://github.com/yhk1038/claude-code-gui-jetbrains/issues/218)

## 새로워진 점

채팅 메시지의 **줄 간격을 직접 조절**할 수 있는 설정이 생겼습니다. 기본 줄
간격이 너무 좁아 읽기 불편했다면, 이제 원하는 만큼 넓히거나 좁힐 수 있습니다.

설정 위치는 **설정 → Appearance → Theme** 이며, **글꼴 크기(Font Size)** 바로
아래에 있습니다.

## 화면에서 보이는 것

- **줄 간격(Line Spacing)** 숫자 입력 필드. **1.0 ~ 3.0** 사이 값을 **0.1
  단위**로 입력합니다. 기본값은 기존 채팅과 동일한 **1.6** 이라, 직접 조절하기
  전까지는 아무것도 바뀌지 않습니다.
- 값을 입력하는 즉시 메시지 본문에 **실시간으로 반영**되므로, 새로고침 없이
  가장 읽기 좋은 값을 찾을 수 있습니다.
- 다른 Appearance 설정들처럼 줄 간격도 **전역(User 설정)** 또는
  **프로젝트별(Project 설정)** 로 지정할 수 있습니다. 프로젝트 값을 비워 두면
  전역 값을 따릅니다.

이 설정은 **채팅 메시지 본문 텍스트**(문단·리스트 항목)에 적용됩니다. 코드
블록은 코드 정렬이 흐트러지지 않도록 자체 고정 간격을 유지합니다.

## 동작 방식

- 값은 표준 CSS `line-height` 배수입니다. 웹뷰가 이 값을 문서 루트의
  `--chat-line-height` CSS 변수로 적용하고, 채팅 메시지 스타일이 이 변수를
  읽습니다. 값이 없을 때는 기존 `1.6` 으로 폴백하므로, 기존 설치 환경도
  이전과 완전히 동일하게 렌더링됩니다.
- 이 설정은 다른 앱 설정(글꼴 크기, 테마 등)과 함께 저장되며, 백엔드에서
  1.0 ~ 3.0 범위를 벗어나지 않도록 검증합니다.

설정 + 웹뷰 변경이므로, 플러그인만 업데이트하면 바로 사용할 수 있습니다.
